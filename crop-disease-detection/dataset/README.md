# Dataset Directory

Place your dataset here in a **folder-per-class** structure.

## Recommended: PlantVillage Dataset

Download from [Kaggle](https://www.kaggle.com/datasets/emmarex/plantdisease) and extract into `data/`:

```
dataset/data/
├── Apple___Apple_scab/
│   ├── image001.jpg
│   └── ...
├── Apple___Black_rot/
│   ├── image001.jpg
│   └── ...
└── ... (38 class folders)
```

Each subdirectory name becomes the class label.
